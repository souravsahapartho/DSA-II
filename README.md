This repository contains implementations, laboratory exercises, and assignments for the **Data Structure and Algorithms II** course. The project focuses on advanced algorithmic strategies, including Divide and Conquer, Greedy algorithms, Dynamic Programming, and Graph theory using C++.

## üìÅ Repository Structure

### 1. Lab Exercises
Practical implementations from weekly laboratory sessions:
* **Lab 01 & 02**: Fundamental C++ review, including input/output operations, frequency counting, and object-oriented concepts with Student and Employee classes.
* **Lab 03**: Recursion-based problems such as Array Sum, the "Make Sandwich" logic, and the N-th Stair problem.
* **Lab 05**: Divide and Conquer strategies, featuring Find Max-Min and the Maximum Subarray problem.
* **Lab 06**: Greedy Algorithms, including the Activity Selection Problem, Fractional Knapsack, and the Greedy Coin Change approach.
* **Lab 07**: Dynamic Programming implementations for the 0/1 Knapsack and optimal Coin Change problems.
* **Lab 08 & 09**: Advanced Graph Algorithms, including Dijkstra's Shortest Path and Minimum Spanning Tree (MST) algorithms like Prim's and Kruskal's.

### 2. Assignments
* **Assignment 01**: A collection of advanced algorithmic problems and solutions (Files: `0112330638_1.cpp` through `0112330638_4.cpp`).
